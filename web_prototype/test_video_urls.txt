## How to Test

### Using Web Interface:
1. Go to https://vision.nexaratech.io
2. Drag and drop your video
3. Click "Analyze Video"
4. View results

### Using cURL:
curl -X POST https://vision.nexaratech.io/api/predict \
  -F "file=@your_video.mp4"

### Using Python:
import requests
url = "https://vision.nexaratech.io/api/predict"
files = {"file": open("your_video.mp4", "rb")}
response = requests.post(url, files=files)
print(response.json())
